## [Techniques from this course](https://www.youtube.com/watch?v=FcxjCPeicvU&t=27326s)

**Every completed section has a branch**

## How to run application

```
$ git clone
$ cd MERN-Tutorial/server
$ npm i
$ npm start


$ cd MERN-Tutorial/frontend
$ npm i
$ npm start

```
